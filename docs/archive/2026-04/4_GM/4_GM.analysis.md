## ❌ 발견된 버그 (수정 완료)

### B1 — `launchQuest` 내 `minSlots` 체크 미수정 [Critical → Fixed]
- **문제**: 사용자 요청으로 `canLaunch >= 1` 변경했으나 `launchQuest()` 내부는 `quest.minSlots` 체크 그대로
- **영향**: q4(minSlots:2)+는 1명으로 UI 버튼 활성화되어도 실제 파견 불가
- **수정**: `slots.length < quest.minSlots` → `slots.length < 1`

### B2 — `trap_disarm` 레벨업 미반영 [Medium → Fixed]
- **문제**: 레벨업 시 `stats.함정해제` 증가하나 `trap_disarm` 필드 미갱신
- **영향**: `calcMercDeathRisk`, `calcSuccessRate`는 `trap_disarm` 직접 참조 → 레벨업해도 함정 성능 고정
- **수정**: 훈련소 레벨업 + 퀘스트 레벨업 두 블록 모두에 `trap_disarm: m.trap_disarm + sb * 2` 추가

### O1 — 파견 중 용병 식량 이중 계산
