## 코드 연결

- `src/assets/uiIcons.ts`에서 UI 아이콘을 한 번에 export
- `src/App.tsx` 상단 자원바의 Day, 금화, 명성, 수정 표기를 이미지 아이콘으로 교체
- 최종 아이콘은 256x256 투명 PNG로 경량화

## 생성 기준

- 기존 2D 손그림풍 판타지 톤 유지
- 작은 크기에서도 읽히는 두꺼운 실루엣
- UI 오버레이에 쓰기 위한 투명 PNG
- 텍스트, 숫자, 로고, 워터마크 없음
