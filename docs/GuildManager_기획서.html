아이언홀드 용병단 길드

중세 암흑 판타지 × 길드 경영 시뮬레이션 — 4_GM

⚛ React 18 + TypeScript 🏰 1600×900 Fixed Canvas 📜 23 퀘스트 🗡 15 무기 👥 5 직업 × 4 종족 💾 3슬롯 자동저장

🌑 세계관

아이언홀드

험준한 산맥과 마족의 영토가 맞닿은 요새 도시. 전쟁, 야수, 함정, 부패한 귀족들이 끊임없이 용병을 필요로 한다.

플레이어는 도시 변두리의 낡은 건물을 사들여 용병단 길드를 세웠다. 아직 이름도 없는 작은 길드지만, 언젠가 이 땅에서 가장 강력한 용병단이 될 것이다.

🏚→🏰
초기 상태

낡은 건물 한 채. 용병 3명. 금화 500G. 식량 80.

🐉
최종 목표

드래곤 토벌 — 명성 100 이상, S급 파티 구성.

⭐
성공 척도

명성(Fame)으로 길드 레벨 상승. Lv5가 되면 모든 계약 해금.

🔄 핵심 게임 루프

① 용병 도착
병영 레벨에 따라 N일마다 용병들이 길드 문에 찾아옴. 고용(비용) or 거절. 🔄 150G 일반 새로고침 / 💎 5수정 고급 새로고침 (B급 이상 보장, A/S 확률 대폭 상승).
② 계약 수주
퀘스트 풀에서 계약 선택. 길드 레벨과 홀 레벨에 따라 표시 퀘스트 수·난이도 범위 변동.
③ 파티 구성
클릭 or 드래그로 용병을 슬롯에 배치. 속성 일치 시 ✦ 보너스 표시. 케미 점수 실시간 표시.
④ 실시간 파견
파견 중에도 다른 퀘스트 동시 수행 가능. 전력 초과 시 클리어 시간 자동 단축. 타이머 완료 후 성공/실패 판정.
⑤ 보상 수령
금화·식량·명성·경험치 획득. 사망 시 등급별 장례 비용 차감. 생존자에게 컨디션/호감도 페널티 적용.
⑥ 길드 성장
금화로 건물 건설/업그레이드. 명성으로 길드 레벨 상승. 용병 무기 업그레이드로 전력 강화.

🚶 용병 시스템

핵심 스탯

⚔️
공격력

전투력 계산 핵심. 퀘스트 성공률·사망률 영향.

🔧
함정해제

도적·궁수 전용. 함정 퀘스트 필수 스탯.

💚
생존율

사망 위험 감소. 장기 원정일수록 중요.

🤝
협조성

파티 시너지에 영향. 평균 협조성이 높을수록 사망률 감소.

실효 전력 공식

effPower(m) = (m.power + weaponPowerBonus) × ageMult × (0.4 + 0.6 × condition/100) × favMod ageMult: age ≥ 55 → 0.0 (파견 불가) age ≥ 38 → 1.0 - (age - 38) / 34 × 0.5 (최대 -50%) age < 38 → 1.0 favMod = 1 + (favorability - 50) / 500 // 범위: 0.9 ~ 1.1

레벨업 시스템

경험치 → 레벨업 : expToNext = level × 100 (Lv1→100XP, Lv2→200XP …)
레벨업 시 모든 전투 스탯 +1, 전력(power) +3 상승. 훈련소 배치 시 매일 XP 자동 획득 (레벨에 따라 1/3/6 XP).

캐릭터 스프라이트 시스템 (MercAvatar)

8개 스프라이트 시트에서 크롭한 40종 WebP 스프라이트가 종족×성별×직업 조합으로 자동 매핑된다.
키: {종족}_{성별}_{직업} (예: 인간_남_전사). 총 용량 ~224KB. sharp로 120×160px 크롭·리사이즈.
등급별 CSS drop-shadow 글로우: S=핑크, A=골드, B=초록. 원형 클립+SVG 링 방식에서 실제 스프라이트 이미지로 교체.

⌛ 나이 시스템

게임일은 실제 자정(midnight)마다 +1 증가. 용병은 게임일 경과에 따라 나이가 들며, 나이가 많을수록 레벨업 시 능력치 상승률이 낮아진다. 노쇠·파견 불가 개념은 없음.
나이레벨업 능력치 배율비고

30세88%성장 소폭 감소

🧪 케미 점수 (Chemistry Score)

파티 내 용병들의 협조성·이기심·시너지·종족·나이·성별을 종합해 0~100점의 케미 점수를 산출. 성공률과 사망 위험에 직접 반영된다.

calcChemistryScore(party): base = avgCoop × 0.55 + (100 - avgEgo) × 0.35 + (synAvg - 1) × 50 - (highEgoCount - 1) × 18 // 이기적 용병 2명+ 시 충돌 패널티 [종족 다양성] 단일 종족: score = base × 0.65 + 70 × 0.35 // 극단 줄이고 평균으로 수렴 2종족 혼합: +9 | 3종족: +18 | 4종족: +27 // 다양성 시너지 [나이 유사성] 편차 ≤4세: +10 | ≤9세: +5 // 비슷한 또래 보너스 [성별 혼합] 남녀 혼성: +8 // 균형잡힌 파티 보너스 → clamp(0, 100)
케미 구간등급성공률 보정사망위험 배율
80~100최고+10%×0.80
60~79양호+5%×0.90
40~59불안±0%×1.00
0~39최악-8%×1.20
케미 점수 바는 용병을 2명 이상 배치했을 때 퀘스트 카드 하단에 실시간 표시된다.

✨ 속성 시스템

용병 속성과 퀘스트 속성의 관계에 따라 전력·성공률·생존율이 달라진다. 속성별 기계적 효과는 없으며, 상성 관계만 적용된다.

상성 관계표

관계조건효과
✦ 일치용병 속성 = 퀘스트 속성생존율 +50% (생존 스탯에 직접 가산)
⬆ 상성 유리용병이 퀘스트 속성에 강함effPower ×1.20 · 성공률 +10%
⬇ 상성 불리퀘스트 속성이 용병에 강함effPower ×0.85 · 성공률 -5%
— 중립그 외보정 없음

상성 사이클

삼각 사이클: 🔥 불 → 🌿 자연 → 🧊 얼음 → 🔥 불 (강함 방향)
빛/암흑: ✨ 빛 ↔ 🌑 암흑 (서로 상성 불리)
사망위험 보정: 일치 ×0.50 · 유리 ×0.85 · 불리 ×1.15
암흑 + 함정 퀘스트: 암흑 속성 용병 1명당 추가 성공률 +6%

⭐ 패시브 & 등급 성장 시스템

용병은 등급에 따라 패시브 슬롯이 늘어난다. 10레벨마다 등급이 오르고 새 패시브가 랜덤 배정된다. 특정 패시브 조합은 시너지 보너스를 추가로 제공한다.

등급별 패시브 슬롯

등급DCBAS
패시브 슬롯1개2개3개4개5개

등급 업 조건

10레벨마다 등급 상승: Lv10 → C급, Lv20 → B급, Lv30 → A급, Lv40 → S급 (D 시작 기준)
이미 S급이거나 시작 등급이 높은 경우 해당 단계는 건너뜀. 등급 상승 시 패시브 1개 랜덤 추가 배정.

패시브 효과 종류

능력치형

  • 전력 보너스 (+N)
  • 공격력 보너스 (+N)
  • 함정해제 보너스 (+N)
  • 생존율 보너스 (+N)
  • HP 보너스 (+N)

특수형

  • 사망위험 감소 (deathRiskMod)
  • 경험치 획득 증가 (xpMod +%)
  • 퀘스트 성공 시 사기 보너스
  • 길드 명성 추가 획득

패시브 시너지

특정 패시브 두 가지를 동시에 보유하면 시너지 보너스가 자동 발동된다. 시너지는 능력치 합산에 추가 반영된다.

시너지 확인: 용병 상세 모달 하단 "패시브 시너지" 섹션에서 보유 시너지와 효과를 표시.

⚔ 정기 원정

타 길드(NPC)와 전력을 겨루는 콘텐츠. 파견한 용병은 사망 위험 없이 6시간 후 귀환하며 순위에 따라 금화·명성·수정·장비를 획득한다. 귀환 후 12시간 쿨다운.

📊 전투 공식

성공률 계산

// 기본 성공률 base = min(95, powerRatio × 75 + 10) // powerRatio = totalEffPower / quest.difficulty // 직업 보너스 성직자 포함 → +8% 전사 포함 → +3% 도적 포함 (함정·고컨디션소모 퀘스트) → +10% // 속성 일치 보너스 (용병마다) 불=+13%, 얼음=+8%, 번개=+9%, 자연=+10%, 암흑=+11%, 빛=+14% 암흑 + trapFocus 퀘스트: 추가 +8%/명 trapFocus 퀘스트 totalTrap≥80: +10%, ≥50: +5% // 슬롯 채움 페널티 fillRatio < 0.5 → -15% fillRatio < 0.75 → -5% // 컨디션 페널티 (개인별) condition < 10 → 파견 불가 (⛔) condition < 25 → -20% condition < 50 → -10% // 케미 점수 보정 chem ≥ 80 → +10% chem ≥ 60 → +5% chem < 40 → -8% 최종 = clamp(5, 95)

사망 위험 계산

// 기본: quest.deathRisk (퀘스트별 기본 사망률) // 0. 전력 비율 보정 powerRatio < 0.4 → ×5.0 (학살) powerRatio < 0.6 → ×3.0 powerRatio < 0.8 → ×1.8 powerRatio < 0.95 → ×1.2 powerRatio ≥ 1.5 → ×0.6 // 1. 퀘스트 종류별 요구 스탯 함정형 (conditionDrain≥20): trapFactor = max(0.5, 1.6 - trapStat/35) 전투형 (deathRisk≥0.12): atkFactor = max(0.55, 1.45 - atkStat/55) 장기형 (duration≥4): durFactor = max(0.5, 1.35 - survStat/75) // 2. 개인 생존율 × max(0.28, 1 - survStat/120) // 3. 파티 구성 성직자 포함 → ×0.65 전사 포함 → 비전사용병에게 ×0.82 도적 포함 → 함정퀘에서 ×0.78 // 4. 소규모 파티 페널티 (인원 < 3) backupPenalty = 1.0 + (1 - partySize/3) × (1.2 - survNorm×0.9) // 5. 협조성 보정 coopFactor = max(0.72, 1.25 - avgCoop/65) // 6. 직업-퀘스트 미스매치 마법사 (저위험·힐러 없음) → ×1.18 성직자 → ×0.72 전사 → ×0.88 // 7. 파티 내 상대 전력 (약한 유닛이 먼저 쓰러짐) relStrength = mercEff / partyAvgEff // 8. 케미 점수 배율 chem ≥ 80 → ×0.80 chem ≥ 60 → ×0.90 chem < 40 → ×1.20 // 9. 컨디션 배율 (개인) condition < 10 → ×4.0 condition < 25 → ×2.5 condition < 50 → ×1.5

🏆 성공 요인 분석

아이언홀드 용병단 길드가 플레이어에게 오래 기억될 게임이 될 수 있는 핵심 요인을 분석한다.

① 게임 디자인 강점

핵심 루프
명확하고 중독적인 고용 → 파견 → 성장 루프

용병 고용, 파티 구성, 퀘스트 파견, 보상 수령, 건물 업그레이드로 이어지는 루프는 짧고 명확하다. 플레이어는 매 결정마다 즉각적인 피드백을 받고, 다음 행동으로 자연스럽게 유도된다. "just one more quest" 효과가 강하게 발동한다.

감정적 애착
나이 시스템이 만드는 용병에 대한 감정적 유대

용병이 30일마다 늙고 언젠가 파견 불가 상태가 된다. 플레이어는 성장시킨 S급 용병을 잃지 않기 위해 수정을 아끼고 나이 동결을 고민한다. 단순한 숫자가 아닌 "살아있는 캐릭터"로 느끼게 만드는 핵심 장치다.

전략 깊이
속성 × 직업 × 케미 × 나이 — 다층적 파티 빌딩

단순히 강한 용병을 넣는 것만으로는 최적이 아니다. 속성 일치 보너스, 직업 역할(탱커/힐러/트래퍼), 케미 점수, 나이별 전력, 무기 종족 친화까지 고려해야 한다. 초보자도 진입할 수 있지만 고수에게는 끝없는 최적화 여지가 있다.

긴장감
실시간 파견 + 퍼마데스 요소의 조합

파견 중 타이머가 실시간으로 흐르고, 완료 후 사망 판정이 실행된다. S급 용병이 사망하면 장례 비용 1,000G가 빠져나가고 생존자의 사기까지 떨어진다. 이 위험성이 파티 구성 한 번 한 번을 긴장되게 만든다.

스토리 연출
퀘스트 체인이 부여하는 서사와 목표 의식

밀수의 그림자, 얼음 아래의 비밀, 어둠의 각성 — 3개의 체인은 퀘스트에 이유를 부여한다. 체인 완료 시 연출되는 스토리 모달은 단순 반복 수행을 "이야기의 일부"로 바꿔준다. 플레이어는 다음 체인을 보기 위해 계속 플레이한다.

동적 시간
전력 초과 시 시간 단축 — 성장의 체감을 극대화

초반에 30분 걸리던 퀘스트가 강해진 파티로 9분 만에 끝난다. 성장이 숫자가 아닌 "실제 기다리는 시간의 단축"으로 체감된다. 번개 속성 용병 육성의 동기도 여기서 나온다.

② 기술·배포 강점

🌐 제로 설치 진입 장벽

단일 HTML 파일 하나로 브라우저에서 즉시 실행된다. 설치, 가입, 인터넷 연결 불필요. 공유와 배포가 극도로 쉽다. "이거 파일 하나야?" 라는 반응이 입소문을 만든다.

💾 외부 서버 없는 완결성

localStorage 3슬롯 저장으로 서버 비용 없이 완전한 저장 시스템 제공. 서비스 종료 걱정 없이 영구 플레이 가능. 인디 게임의 신뢰성 확보에 결정적.

🖥 데스크톱 앱 품질

Electron 패키징으로 EXE 더블클릭 실행 지원. 웹 기반이지만 사용자 경험은 네이티브 앱과 동일. 스팀 비공개 배포, 직접 공유 모두 가능.

📐 1600×900 고정 해상도

고정 캔버스로 UI 일관성이 완벽하게 보장된다. 반응형 레이아웃의 깨짐 없이 설계한 대로 정확하게 표시된다. 디자인 품질이 높아 보이는 효과.

③ 시장 포지셔닝

🎮
타깃 장르

길드 경영 + 로그라이트 요소. Darkest Dungeon의 긴장감 + Recettear의 경영 감각을 웹에서 구현.

👤
핵심 유저층

전략 경영 게임 팬 / 인디 게임 지지층 / 짧은 플레이 세션을 선호하는 직장인 게이머.

✅
차별점

나이 시스템 + 케미 점수 + 속성 체계가 결합된 파티 빌딩은 동종 장르에서 보기 드문 조합.

④ 장기 리텐션 요소

단기 (1~3시간): 첫 길드 레벨업, 첫 퀘스트 체인 완료, 첫 S급 용병 고용
중기 (3~10시간): 건물 풀업, 무기 T3 달성, 나이 동결 전략 완성
장기 (10시간+): 드래곤 토벌 (명성 350 필요), 완벽 파티 구성, 체인 3종 클리어
뉴 게임 리플레이어빌리티: 랜덤 용병 생성 (속성·종족·등급·성별·이름)으로 매 회차 다른 파티 조합이 나온다. 체인 순서 선택, 건물 업그레이드 우선순위에 따라 플레이스타일이 달라진다.
