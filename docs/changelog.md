# 길드 매니저 — 작업 이력

현재 버전: **v1.1.42**

---

## 세션 작업 요약 (v1.1.0 → v1.1.42)

### 1. 코드 구조 리팩토링

단일 `App.tsx` 파일(God Component)을 기능별 모듈로 분리.

| 파일 | 역할 |
|------|------|
| `src/data/weapons.ts` | 무기 데이터 및 `DEFAULT_WEAPON` |
| `src/data/quests.ts` | 전체 퀘스트 데이터 (`ALL_QUESTS`) |
| `src/data/buildings.ts` | 건물 상수 및 헬퍼 함수 |
| `src/constants.ts` | 표시용 상수 (아이콘, 색상, 등급 등) |
| `src/types.ts` | 공용 타입 정의 |
| `src/utils/quest.ts` | 퀘스트 계산 유틸 |
| `src/utils/format.ts` | 포맷/표시 헬퍼 |
| `src/hooks/useGameLoop.ts` | 퀘스트 완료 루프 훅 |
| `src/components/MercAvatar.tsx` | 절차적 SVG 초상화 컴포넌트 |

- 56×56 SVG 기반 캐릭터 아이콘
- 종족·성별·직업·등급에 따라 색상/형태 자동 생성
- 게이트 카드, 용병 카드, 상세 패널 등 7개 위치에 적용

---

### 4. 나이 시스템

- 30 게임일마다 용병 나이 +1
- **38세 이상**: 능력치 점진적 저하 (최대 -50%)
- **55세 이상**: 파견 불가 (`effPower` 0 반환)
- **나이 동결**: 수정 3개 소비 → 30 게임일(150분) 고정
- 용병 상세 패널에 나이/동결 UI 표시

---

### 5. 케미 점수 (Chemistry)

- `calcChemistryScore(party)` 함수 추가
- `traits.cooperation`, `traits.ego`, `traits.synergy_factor` 기반 계산
- 파티 구성 시 성공률 ±보정, 사망 위험 배율 적용
- 퀘스트 배치 UI에 케미 바 표시

---

### 6. 퀘스트 체인 & 스토리 연출

**체인 1 — 밀수의 그림자** (q3 → q6)
**체인 2 — 얼음 아래의 비밀** (q12 → q15 → q18)
**체인 3 — 어둠의 각성** (q2 → q13 → q8)

- `requiredQuestId` 필드로 선행 퀘스트 완료 조건 게이팅
- 체인 완료 시 보라색 스토리 모달 연출 (제목 + 대사 + 신규 의뢰 알림)
- `completedQuestIds` 상태로 진행 추적 및 세이브 저장

---

### 7. 퀘스트 밸런스

- 5분 이내 단기 퀘스트 5종 추가 (비율 10% → 29%)
- 보상 상한 적용 (~10G/분, 최대 1400G)
- D/C 등급 일당 대폭 인하: D=4G, C=10G (기존 15/25G)
- `MISSION_PAY_PER_DAY`: D=4, C=10, B=45, A=90, S=160

---

### 8. 컨디션 & 사망 페널티 강화

**컨디션 임계치:**
- 50 이하: 성공률 -10%
- 25 이하: 성공률 -20%
- 10 이하: 성공률 -35%, 파견 불가

**사망 발생 시 생존자 패널티:**
- 성공 중 전사: 생존자 컨디션 -15 × 사망 수, 호감도 -5, 사기 -5
- 실패 중 전사: 생존자 컨디션 -20 × 사망 수, 호감도 -3, 사기 -5
- 실패 후 전원 귀환: 컨디션 -5 (패배 피로)
- 전투 결과 로그에 `💀 전사 알림` + `😔 생존자 저하` 메시지 추가

---

### 9. 전투력 초과 시 클리어 시간 단축

- 연속 보간 공식: `ratio 1.0 = 기본`, `ratio 2.0 ≈ 65% 단축`, `ratio 3.0+ = 최대 70% 단축`
- 번개 속성 일치 시 추가 단축
- 퀘스트 배치 UI에 **예상 시간** + **단축량(⚡ -N분)** 실시간 표시

---

### 10. 입단 희망자 UI 개선

- 고용/거절 버튼을 세로 분리 (기존: 가로 22px 한 줄)
- 고용 버튼: 초록 테두리, 높이 26px
- 거절 버튼: 빨강 테두리, 높이 22px
- 카드 높이 108 → 130px

---

### 11. 빌드 스크립트 개선 (`scripts/build-release.js`)

- 빌드마다 구 HTML(`release/`) 및 루트 EXE 자동 삭제
- 새 EXE를 프로젝트 루트에 복사 (실행 접근성 향상)
- 잠금 파일(실행 중 EXE 등) 삭제 실패 시 예외 처리로 건너뜀
